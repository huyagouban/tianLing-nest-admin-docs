# 上传下载

::: tip 提示
业务场景中，会用到上传和下载功能，因此做了通用的上传和下载功能。
:::

::: warning 友情提示
目前项目仅支持了图片上传和视频上传，其他类型的文件上传功能，请自行实现。

下载功能，请自行实现。

功能会逐步完善，敬请期待。
:::
